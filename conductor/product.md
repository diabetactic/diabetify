# Product Guide

## Initial Concept

Diabetactic (Diabetify) is a mobile app for diabetes glucose management built with Ionic/Angular. It allows users to track glucose levels, visualize trends, and manage medical appointments.

## Target Audience

- **Primary Users:** Patients with Type 1 or Type 2 Diabetes who need to track their glucose levels.
- **Specific Pilot Group:** Kids with Type 1 Diabetes who will be using the app for fast virtual appointments with their doctors.
- **Secondary Users:** Healthcare professionals (doctors, endocrinologists) who may review patient data.

## Core Value Proposition

- **Health Management:** specific tools for recording and analyzing glucose readings.
- **Virtual Care:** Optimized for fast, efficient data sharing during virtual medical appointments.
- **Accessibility:** Offline-first architecture ensures data is always accessible and safe.
- **Usability:** Clean, bilingial (English/Spanish) interface with light/dark themes, suitable for younger users and their families.

## Key Features

1.  **Glucose Management:**
    - Log glucose readings manually.
    - View historical logs.
2.  **Dashboard & Analytics:**
    - Visual trends and statistics of glucose levels over time.
3.  **Appointment Management:**
    - Schedule and track medical appointments.
    - Support for fast virtual appointments.
4.  **Offline Capability:**
    - Local data storage (Dexie/IndexedDB) with background synchronization.
5.  **Authentication:**
    - Integration with Tidepool for secure user access.
6.  **Customization:**
    - Dark/Light mode support.
    - Multi-language support (English/Spanish).

## Success Metrics

- Successful synchronization of offline data.
- User engagement with the dashboard (daily active users).
- Reliability of glucose data recording.
- Efficiency and speed of data sharing during virtual appointments.
