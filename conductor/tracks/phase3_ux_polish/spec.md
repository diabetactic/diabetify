# Spec: UX Modernization & Polish

## Goal

Elevate the visual design and user interaction quality to match modern mobile standards.

## Core Features

1.  **ShadCN-Inspired Redesign**
    - Implement new component styles based on ShadCN design system (POC).
    - Refine typography, spacing, and border radius.

2.  **Dark Mode Consistency**
    - Ensure all screens adapt correctly to system dark mode.
    - Verify color contrast meets accessibility standards.

3.  **Haptics & Error Recovery**
    - Add haptic feedback to primary interactions (success, error, warning).
    - Design and implement user-friendly error recovery flows (e.g., offline retry banners).

## Success Metrics

- Visual consistency across all major screens.
- Zero contrast accessibility violations in Dark Mode.
- Haptic feedback present on all key actions.
