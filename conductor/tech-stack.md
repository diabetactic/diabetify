# Technology Stack\n\n## Core Technologies\n- **Programming Language:** TypeScript 5.9\n- **Frontend Framework:** Angular 21\n- **UI Framework:** Ionic 8 (Standalone Components)\n- **Mobile Runtime:** Capacitor 8.0\n- **Styling:** Tailwind CSS 3.4 + DaisyUI 5\n\n## Architecture & Data\n- **Architecture:** Modular, Offline-First, Signal-based State Management\n- **Local Database:** Dexie 4.2 (IndexedDB wrapper)\n- **State Management:** Angular Signals + RxJS (for async streams)\n- **API Communication:** Angular HttpClient with Interceptors\n\n## Testing Strategy\n- **Unit Testing:** Vitest 4\n- **Integration Testing:** Vitest 4 + MSW (Mock Service Worker)\n- **End-to-End (E2E) Testing:** Playwright 1.48\n- **Visual Regression:** Playwright\n\n## Quality Assurance\n- **Linting:** ESLint (Flat Config) with Angular-ESLint\n- **Formatting:** Prettier\n- **Static Analysis:**\n - `knip` (Dead code detection)\n - `dpdm` (Circular dependency detection)\n - `type-coverage` (TypeScript type safety)\n - `stylelint` (CSS/SCSS linting)\n\n## Build & Tooling\n- **Package Manager:** pnpm 10+\n- **Build System:** Angular CLI + Turbo (for task orchestration)\n- **CI/CD:** GitHub Actions (implied from directory structure)
